## STRATEGIES & INSIGHTS

[sai-00001] helpful=0 harmful=0 :: Always determine the underlying accounting event before selecting a tag: shares issued in a business acquisition → BusinessAcquisitionEquityInterestsIssuedOrIssuableNumberOfSharesIssued; cash paid as part of an acquisition → PaymentsToAcquireBusinessesGross; total consideration transferred in the combination → BusinessCombinationConsiderationTransferred1.
[sai-00005] helpful=0 harmful=0 :: When the narrative mentions mandatory convertible preferred stock or other convertible debt instruments, first classify the instrument as convertible debt. Then map the share count to StockIssuedDuringPeriodSharesNewIssues and the price to DebtInstrumentConvertibleConversionPrice1, rather than using generic equity‑sale tags.
[sai-00008] helpful=0 harmful=1 :: When a numeric value denotes the size of a revolving credit facility (e.g., "maximum borrowing capacity of $225.0 million"), map it to the US GAAP tag LineOfCreditFacilityMaximumBorrowingCapacity. When percentages describe loan margin or basis spreads on variable‑rate debt (e.g., "margin spread of 1.00%"), map them to DebtInstrumentBasisSpreadOnVariableRate1.
[sai-00010] helpful=1 harmful=0 :: When a numeric value is mentioned alongside a revolving credit facility, first decide whether the number represents the facility's total size (maximum borrowing capacity) or the amount currently available. Use LineOfCreditFacilityMaximumBorrowingCapacity for the facility size; only use LineOfCreditFacilityCurrentBorrowingCapacity when the text explicitly states the current or remaining amount (e.g., "available", "remaining", "current draw"). This rule should be applied consistently across all similar questions.
[sai-00013] helpful=0 harmful=3 :: When the narrative provides a numeric interest rate (e.g., "8 %"), map it to the tag DebtInstrumentInterestRateStatedPercentage. When it provides interest expense amounts, map them to InterestExpenseDebt. When a percentage describes revenue concentration (e.g., "44 % of revenue"), map it to ConcentrationRiskPercentage1. Always confirm the tag exists in the US GAAP tag list and choose the most specific tag that matches the semantic type (rate, amount, percentage).
[sai-00015] helpful=0 harmful=0 :: When the narrative mentions a par value per share (e.g., "$0.001"), map it to the US GAAP tag CommonStockParOrStatedValuePerShare, which captures the per‑share stated or par value of common stock.
[sai-00016] helpful=0 harmful=0 :: When the text refers to share‑based compensation plan counts, select the specific share‑based compensation tags: use ShareBasedCompensationArrangementByShareBasedPaymentAwardNumberOfSharesAvailableForGrant for language indicating the number of shares currently available for grant (e.g., "no shares available"), and use ShareBasedCompensationArrangementByShareBasedPaymentAwardNumberOfSharesAuthorized for language indicating the total authorized or maximum deliverable shares for a share‑based award.
[sai-00018] helpful=2 harmful=0 :: When an interest rate is described as a benchmark plus a percentage (e.g., "LIBOR plus 4.15%" or "prime + 1.5%"), treat it as a basis spread rather than a fixed stated rate. Map such expressions to the tag DebtInstrumentBasisSpreadOnVariableRate1. Only use DebtInstrumentInterestRateStatedPercentage for rates that are presented as absolute, fixed percentages without reference to a benchmark.
[sai-00023] helpful=0 harmful=0 :: When a debt instrument's narrative mentions a principal (or face) amount, map it to DebtInstrumentFaceAmount; when it mentions the fair value of the debt, map it to DebtInstrumentFairValue. Apply this rule to both convertible and non‑convertible debt instruments.
[sai-00025] helpful=0 harmful=0 :: When the narrative describes adverse or favorable development of loss reserves, prioritize the most specific tag that references prior‑year claims and claim‑adjustment expense. Map such figures to SupplementalInformationForPropertyCasualtyInsuranceUnderwritersPriorYearClaimsAndClaimsAdjustmentExpense rather than using generic loss‑contingency tags.
[sai-00028] helpful=0 harmful=0 :: When the narrative reports the number of RSUs (or other equity‑instrument awards) granted in a period, map it to ShareBasedCompensationArrangementByShareBasedPaymentAwardEquityInstrumentsOtherThanOptionsGrantsInPeriod, which captures equity‑instrument (non‑option) grants within the reporting period.
[sai-00029] helpful=0 harmful=0 :: When the text mentions unrecognized stock‑based compensation expense (e.g., cost of non‑vested awards not yet recognized), select EmployeeServiceShareBasedCompensationNonvestedAwardsTotalCompensationCostNotYetRecognized rather than generic tax‑benefit tags.
[sai-00034] helpful=0 harmful=0 :: When a numeric value is described as the fair value of a debt instrument (especially long‑term debt), select the tag LongTermDebtFairValue. Do not use DebtInstrumentCarryingAmount or the generic DebtInstrumentFairValue tag in this situation; the measurement context (fair value) overrides incidental descriptors such as "carrying amount".
[sai-00036] helpful=0 harmful=0 :: When a numeric percentage represents a statutory corporate federal income tax rate (e.g., "35%" before the Tax Cuts and Jobs Act or "21%" after), map it to the US GAAP tag EffectiveIncomeTaxRateReconciliationAtFederalStatutoryIncomeTaxRate.
[sai-00038] helpful=0 harmful=0 :: When a percentage quantifies the proportion of total revenue, net revenue, or trade accounts receivable attributable to a single customer, distributor, or partner, map it to the ConcentrationRiskPercentage1 tag. This rule applies regardless of whether the narrative mentions "revenue", "net revenue", "trade accounts receivable", or similar financial streams. Verify that the tag exists in the US GAAP list and that no more specific concentration‑risk tag is available.
[sai-00040] helpful=0 harmful=0 :: When the narrative mentions depreciation expense or depreciation of property, plant, and equipment, map the amount to the US GAAP tag Depreciation. When the text refers to amortization of intangible assets (e.g., "amortization expense" or "amortization of goodwill"), map it to the tag AmortizationOfIntangibleAssets. Always verify that the selected tag exists in the provided tag list and prefer the most specific tag that matches the semantic type (expense amount vs. rate).

## COMMON MISTAKES TO AVOID

[err-00002] helpful=2 harmful=0 :: Do not map any share count to CommonStockSharesOutstanding; reserve that tag only for existing outstanding shares, not for shares issued as part of an acquisition.
[err-00003] helpful=0 harmful=0 :: Do not assign purchase‑price figures to BusinessCombinationConsiderationTransferred1 unless the amount represents the overall consideration; cash components of a purchase should be tagged PaymentsToAcquireBusinessesGross.
[err-00006] helpful=0 harmful=0 :: Do not treat convertible preferred stock transactions as ordinary equity sales; avoid tags like SaleOfStockNumberOfSharesIssuedInTransaction and SaleOfStockPricePerShare. Instead use convertible‑debt specific tags that capture issuance and conversion price.
[err-00011] helpful=0 harmful=0 :: Do not default to LineOfCreditFacilityCurrentBorrowingCapacity when the sentence only mentions the credit‑facility limit. Even if the wording includes qualifiers like "may be less than" or "availability may be less than $X", treat the $X as the facility's maximum size unless the text explicitly describes the amount presently available.
[err-00020] helpful=0 harmful=0 :: Do not apply the generic rule that maps any numeric interest rate to DebtInstrumentInterestRateStatedPercentage when the rate is expressed as a benchmark plus a spread. Misclassifying a benchmark‑plus rate as a stated rate leads to incorrect tagging; instead, use DebtInstrumentBasisSpreadOnVariableRate1 for such spreads.
[err-00022] helpful=0 harmful=0 :: Do not assign amounts described as the "borrowing base" to LineOfCreditFacilityMaximumBorrowingCapacity; they represent the current borrowing capacity and should be tagged accordingly.
[err-00027] helpful=0 harmful=0 :: Do not default to generic loss‑contingency tags (e.g., LossContingencyEstimateOfPossibleLoss) for reserve development amounts. The correct tag is the supplemental‑information tag that explicitly references prior‑year claims and claim‑adjustment expense.
[err-00030] helpful=0 harmful=0 :: Do not use ShareBasedCompensationArrangementByShareBasedPaymentAwardNumberOfSharesAuthorized for RSU grant counts; that tag refers to total authorized shares, not to grants made in the period.
[err-00031] helpful=0 harmful=0 :: Do not map unrecognized stock‑based compensation expense to UnrecognizedTaxBenefits; the correct tag is EmployeeServiceShareBasedCompensationNonvestedAwardsTotalCompensationCostNotYetRecognized.
