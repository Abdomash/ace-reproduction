# Context

This directory contains user-created planning notes, implementation notes, experiment logs, and paper context for humans and AI agents.

The files here are not raw experiment outputs. Raw run artifacts belong in `results/`; derived analysis products belong in `analysis/outputs/`.

## Contents

- `API.md`, `APPWORLD.md`, `CODE-CHANGES.md`, `ENVIRONMENT.md`, `EXPERIMENTS.md`, `JOURNAL.md`, `NOTES.md`, `PLAN.md`: working context for the reproduction.
- `papers/`: paper notes.
- `runners/`: runner-specific notes.
