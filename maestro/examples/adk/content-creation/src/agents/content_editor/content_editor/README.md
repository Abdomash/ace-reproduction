# Content Editor Agent
*from [a2a-samples](https://github.com/a2aproject/a2a-samples/tree/883c8906a60cb48e43599db0480d2217bd02c320/samples/python/agents/content_planner)*

Given a high-level description of the content that's needed, this sample agent creates a detailed content outline. This agent is written using the Google Agent Development Kit (ADK) and the Python A2A SDK.

## Prerequisites

- Python 3.10 or higher
- [UV](https://docs.astral.sh/uv/)
- Access to an LLM and API Key

## Building Docker

```bash
docker build -t a2a/content_editor .
```
